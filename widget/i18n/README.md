# Quick Note Widget for amoCRM

Кнопка для сырых заметок в карточке сделки.

## Структура виджета

```
widget/
  manifest.json        # Метаданные и конфиг виджета
  script.js           # Основной JS-код виджета
  i18n/
    ru.json          # Локализация (русский)
  README.md           # Этот файл
```

## Основные фичеры

- Кнопка "Добавить заметку" в карточке сделки
- Сохранение текстовой заметки в ленте сделки
- Настройка текста в settings
- Статусы: лоадинг, сукцесс, ошибка

## Использование

1. Откройте карточку сделки
2. Найдите виджет "Quick Note"
3. Нажмите "Добавить заметку"
4. Заметка с текстом по умолчанию сохранится в ленте
